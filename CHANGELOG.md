# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-02

### Added
- NestJS backend workspace bootstrap with TypeScript and Vitest.
- Prisma schema, migration, seed, and runtime Prisma module.
- Auth, navigation, admin, dashboard, AI, questions, and sessions modules.
- Shared infrastructure: guards, decorators, filters, interceptors, pipes, Redis integration.
- Unit and API-level test coverage scaffolding.
- CI workflow for lint, test, typecheck, and build.
- Project setup and environment documentation.
